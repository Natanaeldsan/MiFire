
# MiFire

Software para análise de estruturas em situação de incêndio.

## Módulos

- Curvas de incêndio (ISO 834, Hidrocarbonetos, Exterior, Paramétrica)
- TRRF (tempo requerido de resistência ao fogo)
- Análise térmica de aço
- Dimensionamento de elementos de aço
- Seção residual de madeira

## Normas utilizadas

- NBR 14432
- NBR 14323
- NBR 8800
- NBR 7190
- Instruções Técnicas do Corpo de Bombeiros

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
